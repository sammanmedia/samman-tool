# samman-tool
Membuat aplikasi Alat untuk Distro Linux
==================================
=== tool yang tersedia ===
1. Penghapus virus adalah menghapus semua file berada di forder dengan format di bawah ini
   ("pif" "vbs" "jar" "exe" "scr" "bat" "js" "cmd" "lnk")
   di bantai secara realtime tidak akan berhenti scanning karena realtime untuk berhenti scanning bisa
   tekan terminal CTRL + C
   untuk + TELE itu jika ketika ada virus nya maka akan melaporkan ke telegram bot follow aja botnya di @samman_bot
2. Samba, adalah aplikasi Lan Scanner yang terbuka samba nya, bisa melihat Ip address, UP/DOWN, Hostname, dan Macc address,
   scanning hanya 1 kali
3. Ping canggih, adalah aplikasi untuk ngeping jaringan local mirip samba
4. IP saya, adalah aplikasi mendeteksi Ip saya, Ip Publik, Ip Local, Negara, Isp, Asn, Proxy (ya / tidak), Bendera. dan di simpan melalui folder log ip-saya.txt

=======

cara membuka aplikasinya
1. Extract file nya
2. chmod +x *.sh
3. jalankan ./menu.sh
4. Selesai
